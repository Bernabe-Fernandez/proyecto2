El Back-End en el desarrollo Web

El backend se puede definir como un termino utilizado para hacer referencia a la capa de acceso a datos de un software, es decir a toda el área lógica de una pagina web, es la parte del desarrollo web que nos permite hablar acerca de la arquitectura interna de los sitios web, asegura que los elementos desarrollen su funcionalidad de forma correcta un ejemplo de la funcionalidad del back-end sería la comunicación con el servidor, el sistema de gestión de pedidos, inventario y procesamiento de suministros. Es la parte de una pagina web, que no es visible para el usuario, y la cual no incluye elementos gráficos a diferencia de su contraparte el front-end.
